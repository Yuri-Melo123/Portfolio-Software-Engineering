# Controle de Estoque - Projeto com Tkinter

## Funcionalidades:
- Login com autenticação e perfis (admin/comum)
- Criptografia de senha (SHA256)
- Cadastro de produtos com controle de mínimo
- Atualização de estoque
- Visualização com alerta de baixo estoque
- Cadastro de novos usuários (somente admin)
- Gerenciamento de quadro Kanban

## Requisitos:
- Python 3
- Interface Tkinter (já incluída no Python)
